# Navigator-Main – IA de Enemigos 3D

Proyecto académico centrado en el desarrollo de **inteligencia artificial básica para enemigos** en un entorno 3D utilizando **Unity y C#**.

## Descripción
El objetivo del proyecto es implementar enemigos con distintos **patrones de movimiento y comportamiento**, capaces de detectar al jugador y reaccionar según su estado.

## Funcionalidades principales
- Patrones de movimiento de enemigos.
- Detección del jugador.
- Cambio de comportamiento según estados (patrulla, persecución).
- Gestión básica de IA orientada a videojuegos.

## Tecnologías utilizadas
- Unity
- C#
- Programación orientada a objetos
- Sistemas de estados

## Aprendizajes
- Diseño de comportamientos para enemigos.
- navMesh para movimiento tanto de Player como de enemigo
- Lógica de IA aplicada a videojuegos.
- Organización del código para sistemas de comportamiento.
